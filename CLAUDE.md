# The Vow Edit

## Design / Frontend work — ALWAYS use the taste skill

Before writing or editing ANY frontend code (pages, components, layouts,
styling, landing pages, redesigns, UI), you MUST invoke the
`design-taste-frontend` skill (installed at
`.claude/skills/design-taste-frontend/SKILL.md`) and follow its guidance.

This is non-negotiable for this project: every UI/visual task starts by
loading that skill. Do not produce frontend code without it.
